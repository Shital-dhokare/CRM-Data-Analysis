# CRM Analytics Dashboard Project

## Project Overview
This project focuses on CRM (Customer Relationship Management) analytics using SQL, Excel, Tableau, and Power BI.

The project analyzes lead generation, customer conversion, opportunity performance, and business insights through interactive dashboards and KPI tracking.

It helps understand customer behavior, lead conversion trends, and overall business performance using data-driven analysis.

---

## Objectives
- Analyze lead generation and conversion trends
- Track sales and customer performance
- Monitor KPI metrics and business growth
- Identify high-performing sources and regions
- Build interactive dashboards for decision-making

---

## Tools Used
- SQL
- Excel
- Tableau
- Power BI

## Key Insights
- Win ratio is lower compared to loss ratio
- Revenue is mainly generated from successful won deals
- Certain stages show higher opportunity drop-offs
- Monthly trends help track sales performance over time
- Loss reason analysis helps identify improvement areas
